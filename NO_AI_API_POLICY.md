# No AI API Policy — ML Lab Enterprise

**Maintainer:** Adewale Samson Adeagbo · HMG Concepts

ML Lab Enterprise intentionally does not use paid AI APIs.

## Why

Paid AI APIs are not cost-effective for many schools, beginner learners, community programmes and free EdTech pilots.

## What the platform uses instead

- Curated lessons
- Rule-based Local Mentor
- Static templates
- Local calculators
- Deterministic checklists
- Browser localStorage
- Downloadable/exportable files

## What is not used

- OpenAI API
- Gemini API
- Claude API
- Paid AI chatbots
- Backend AI inference
- Secret API keys

## Data Protection Benefit

Because no AI API is used, learner content is not sent to external AI providers.

## Future Note

If an institution later wants AI API features, that should be treated as a separate paid extension with budgeting, consent, privacy review and admin controls.
