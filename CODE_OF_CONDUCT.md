# Code of Conduct — ML Lab Enterprise

ML Lab Enterprise is built by Adewale Samson Adeagbo / HMG Concepts to support respectful, accessible and practical learning.

Participants should:

- Be respectful
- Explain clearly
- Avoid gatekeeping
- Support beginners
- Credit original work
- Protect learner privacy
- Avoid harmful or discriminatory use of ML
