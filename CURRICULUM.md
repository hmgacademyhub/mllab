## Curriculum Goal

This curriculum moves a beginner data scientist from first principles to portfolio-ready competence in practical machine learning.

It is designed to be:

- Beginner-friendly
- Progressive
- Detailed
- Practical
- Project-based
- Free-tool based
- No-AI-API dependent
- Suitable for self-study, school cohorts, teacher training and HMG Concepts programmes

---

## Learning Philosophy

The curriculum follows Adewale Samson Adeagbo's teaching approach:

1. Start with clear problem framing.
2. Explain concepts in plain English.
3. Show working code.
4. Practise through active recall.
5. Evaluate honestly.
6. Document limitations.
7. Deploy with free tools.
8. Keep humans accountable for high-impact decisions.

## Assessment Structure

| Stage | Evidence of Competence |
|---|---|
| Beginner Ready | Can split data, preprocess, train baseline and explain leakage |
| Intermediate Ready | Can build pipelines, evaluate models, tune hyperparameters and explain metrics |
| Advanced Ready | Can interpret models, tune thresholds, handle NLP/time data and monitor drift |
| Portfolio Ready | Can deploy/document a complete project with limitations and responsible ML notes |

## Capstone Options

Learners choose one:

1. EdTech Early-Warning System
2. Customer Churn Retention Model
3. Burnout or Score Regression Predictor
4. Fake News / Text Classifier
5. Learner Segmentation with Clustering

Each capstone must include:

- Problem statement
- Data dictionary
- EDA summary
- Leakage audit
- Pipeline
- Baseline model
- Improved model
- Evaluation metrics
- Error analysis
- Interpretation
- Limitations
- Deployment or deployment plan
- README/model card
