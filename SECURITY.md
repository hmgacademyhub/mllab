## Supported Deployment Model

ML Lab is a static, offline-first learning platform using free tools:

- HTML, CSS and vanilla JavaScript
- Browser localStorage
- GitHub Pages / Cloudflare Pages / Netlify
- No backend
- No database
- No AI API

## Security Principles

1. Do not store sensitive personal data in the platform.
2. Do not commit API keys, passwords or private files.
3. Use HTTPS deployment links only.
4. Use the Backup Center before clearing browser data.
5. Review external links before institutional deployment.
6. Use GitHub Issues for transparent bug reporting.

## No AI API Exposure

The platform does not send learner data to OpenAI, Gemini, Claude or any paid AI API. The Local Mentor is rule-based and local.
