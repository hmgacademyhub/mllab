## Common Support Cases

| Issue | Action |
|---|---|
| Site not opening | Confirm `index.html` is at repository root |
| Progress not saving | Use the local server option or deployed HTTPS URL |
| Old version visible | Hard refresh with Ctrl+Shift+R |
| Need to move data | Use Backup Center export/import |
| Need school deployment | Use Cohort Manager and Governance Center |
| Need no-cost setup | Use GitHub Pages or Cloudflare Pages |
