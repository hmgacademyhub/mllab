## Why ML Lab Exists

ML Lab was built to help learners move from Python and EDA into confident model building with scikit-learn and statsmodels. It reflects Adewale's teaching method: break complex ideas down, explain the why, practise actively and deploy with free tools.

No paid AI API is required. The system uses local, transparent logic so that learners and schools can use it without recurring cost.
