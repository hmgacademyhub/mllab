# Contributing to ML Lab Enterprise

**Maintainer:** Adewale Samson Adeagbo · HMG Concepts

Thank you for improving ML Lab Enterprise.

## Contribution Types

- Fix lesson typos
- Add exercises
- Improve explanations
- Add free-tool deployment notes
- Improve accessibility
- Add curriculum examples
- Report bugs

## Rules

1. Do not remove existing features without discussion.
2. Do not add paid AI API dependencies.
3. Do not commit API keys or secrets.
4. Keep explanations beginner-friendly.
5. Update README, CHANGELOG and CURRICULUM when changing content.
6. Test `index.html` before submitting.
